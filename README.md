# Parqueadero

este programa de consola sirve para llevar la cuenta de espacios disponiobles en un parqueadero de 10 carros de capacidad, existen 2 versiones, una llamada (main.py) en la que la cantidad de vehiculos a ingresar y retirar es solicitada al usuario y otra (new_main.py) en la que el proceso de retiro y ingreso se maneja uno por uno
